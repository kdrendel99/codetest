New languages are being tweaked, reinvented, and published frequently and it's important to be able to stay up-to-date in order to survive as a web developer. Being able to write in different languages is important, but what if you don't know where to start?

You got Javascript!

Javascript is the world's most popular programming language. It's used to program the moving parts in a web browser, however has many other uses too. It's used in everything from building simple websites to big-name companies such as Microsoft, Netflix, and Paypal. Javascript is a great first language to learn and will help you get your foot in the door as well as help you get familiar with learning coding logic.

Ruby is a nice all-in-one package for webdev. It's great to learn for web specifically, however few people use Ruby by itself anymore. So while one could use Ruby as a python replacement, there's more demand elsewhere. If you're interested in web dev and (em)only(/em) webdev, then Ruby is a really marketable language to know. You got Python!

Python is a language that's similar to Ruby. Python is currently an industry standard for many programs. Looking at python vs Ruby, python is a better choice many aspects. Python is widely used by big name companies such as Google, Facebook, Instagram, and Spotify. According to Wikipedia, "It's an interpreted, high-level and general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects."
